# apps/api

Supabase **Edge Functions** are deployed from **`supabase/functions`** at the monorepo root (Supabase CLI). This folder mirrors the bootstrap layout for **documentation and naming** (`ownerrez-oauth`, `ownerrez-webhooks`, `stripe-webhooks`, `analytics-events`).

See [`docs/decisions/adr-0001-repo-structure.md`](../../docs/decisions/adr-0001-repo-structure.md).
